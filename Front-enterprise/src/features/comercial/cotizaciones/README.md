# Componente DetalleCotizacion

Este es el archivo definitivo y activo para mostrar los detalles de una cotización.

## Ubicación del archivo activo:
- `src/features/comercial/cotizaciones/DetalleCotizacion.jsx`

## Ruta donde se utiliza:
- En `src/App.jsx` en la ruta `cotizaciones/:id`

## Notas importantes:
- Este archivo reemplaza al archivo obsoleto en `src/pages/comercial/DetatalleCotizacion.jsx`
- Cualquier modificación debe hacerse en este archivo y no en el duplicado
- La estructura de `features/` es la que se debe seguir para mantener la arquitectura del proyecto
